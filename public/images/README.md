## Image Guidelines

1. **Format**: Use JPG for photos, PNG for graphics with transparency
2. **Optimization**: Compress images before adding (use TinyPNG, ImageOptim, etc.)
3. **Naming**: Use lowercase, hyphen-separated names
4. **Quality**: Use high-quality, professional photography
5. **Consistency**: Maintain consistent aspect ratios within categories

## Adding New Images

Simply drop your images into this folder with the correct names. The application will automatically load them.
